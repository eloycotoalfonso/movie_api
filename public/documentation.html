My movies DOCUMENTATION

This is an server side component aplication for movies. Users will be able to log in, sign up, and get different information about movies, directors and so on. They will also be able to update their personal information and create a list of their favorite movies.
This task is based on the server side, the client-side will be part of the next achievement of the course.

Request URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON object that retrieves data about all the films
Return data about a single movie by title /movies/[Name] GET None A JSON Object retrieving data about a single film
Return data about a genre by name/title /movies/[Genre] GET None A JSON Object retrieving data about one particular genre
Return data about a director /directors/[Name] GET None A JSON Object retrieving data about one particular director
Allow new users to register /users/[New] POST A JSON object holding data about the user like: {
name: "User Name",
age: "age"
}		 A text messsage indicating the user has been registered
Allow users to update their user info (username) /users/[Name] PUT None A text message indicating that the user info. has been updated.
Allow users to add a movie to their list of favorites /movies/[Name] POST A JSON object holding data about the film to add, structured like:
{
title : "Film title",
director : "director Name"
}		 A text message confirming that the movie was succesfully added to favorites.
Allow users to remove a movie from their list of favorites /movies/[Name] DELETE None A text message confirming that the movie was successfully removed from favorites.
Allow existing users to deregister /users/[Name] DELETE None A text message confirming that the user was successfully removed.
